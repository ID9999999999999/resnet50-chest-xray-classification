# Model Card

## Model

ResNet50 pretrained on ImageNet and adapted for binary chest X-ray image classification.

## Task

Classify chest X-ray images into:

- NORMAL
- PNEUMONIA

## Intended Use

Educational machine-learning experimentation and portfolio demonstration.

## Not Intended For

This model is not intended for medical diagnosis, clinical decision-making, or hospital use.

## Dataset

The dataset is not included in this repository.

## Limitations

- Results depend on the dataset split and preprocessing.
- The model may fail on images from different hospitals or devices.
- The project is educational, not clinical.

## Final Status

Portfolio preparation stage.
